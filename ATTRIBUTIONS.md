# Audio attribution

Far Signal currently generates temporary music and some sound effects at run
time in `scripts/audio_manager.gd`.

The following selected sound effects are included in `assets/audio/sfx/`:

- `door_open.ogg`
- `door_close.ogg`
- `player_shot.ogg`
- `enemy_shot.ogg`

Their original source and license still need to be recorded here.

When the procedural placeholders are replaced, record each asset's author,
source URL, license, and any modifications in this file.
